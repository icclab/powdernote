What is powdernote?

Powdernote is a simple CLI-only note-taking application. It is based on a Swift object storage backend.

Why use powdernote?

It is very easy to use and features intuitive commands. The simplicity of the powdernote will make your working days easier by allowing you to access your notes from the command-line.

Who is powdernote for?

Powdernote is for advanced computer-users (power-users) who are constantly working on the command-line and need to keep write down something every now and then.

Installation guide

  1. First things first: download the repository.
  2. Open the Configuration.py file and enter your credentials to the corresponding parameters.
  3. To install powdernote use python setup.py sdist && sudo pip install dist/powdernote-1.0.tar.gz in the directory of powdernote.
  4. Now go ahead and use it. powdernote -h
